Syftet med den praktiska uppgiften den här veckan var att skapa en hemsida som använder javascript för att visa ett meddelande när en knapp klickas.


Här är ett meddelande som enbart visas när man hovrar över knappen
Länk till den teoretiska uppgiften finns uppe i menyn. :)

För skojs skull valde jag även att göra en liten knapp som ändrar mellan ljust och mörkt läge, längst upp i högra hörnet, men eftersom windows har limiteringar för hur dokumenten får hämta information ifrån varandra så fungerar den bara på index-sidan, om hemsidan öppnas lokalt på en dator.
Om den startas via en server så fungerar funktionen så att den byter på alla sidor, och de är alla synkade.
Jag bestämde mig därför för att ladda upp den på ett gratis webhotell, så om du vill kan du kolla där för att se sidan fungera fullt ut.
